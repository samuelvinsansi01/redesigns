Como fazer o deploy

    

      
1

      

        
Crie um repositório no GitHub

        
github.com → New repository → nome: redirect-app → Create

      

    

    

      
2

      

        
Suba os arquivos deste projeto

        
git clone → cole os arquivos → git add . && git commit -m "init" && git push

      

    

    

      
3

      

        
Conecte ao Vercel

        
vercel.com → Add New Project → importe o repo → Deploy (zero config necessário)

      

    

    

      
4

      

        
Adicione o token do TinyURL no Vercel

        
Vercel → projeto → Settings → Environment Variables → adicione TINYURL_TOKEN com o valor do seu token → Save → Redeploy

      

    

    

      
5

      

        
Copie a URL do Vercel e use a aplicação

        
ex: https://redirect-app-xyz.vercel.app → preencha os campos acima e clique em gerar
